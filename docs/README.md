## 🎭 Conceitos Principais

Lembre-se da metáfora do **Hearthstone**:

- **🎴 Deck** = Pipeline (conjunto de etapas)
- **🃏 Card** = Step (uma etapa específica)
- **⚔️ Match** = Execução (uma rodada do pipeline)
- **🏟️ Arena** = Queue (fila de execuções)
